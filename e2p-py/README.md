# py360viewer

Refer this repo [https://github.com/sunset1995/py360convert](https://github.com/sunset1995/py360convert) for convert 360 panorama image

press & move can change angle of view

wheel can change field of view

## Requirement

- numpy
- scipy
- pillow (PIL)
- PyQt5
